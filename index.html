EDENOR · EDESUR · EDELAP

Acometidas, suministros y documentación para distribuidoras

Gestión técnica para obras, obradores, comercios, industrias, edificios y nuevos suministros. Se prepara la documentación eléctrica necesaria y se acompaña el trámite según el requerimiento de cada distribuidora.

EDENOR

Nuevos suministros, aumentos de potencia, pilares, tableros principales, censo de cargas, memorias técnicas y documentación profesional.

EDESUR

Gestión de acometidas, adecuaciones, análisis de potencia, documentación eléctrica y asistencia técnica para presentación.

EDELAP

Trámites para suministros en La Plata y zona de concesión, documentación para grandes usuarios, obradores y proyectos visados cuando corresponda.
